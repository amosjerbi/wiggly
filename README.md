# Wiggly Paint
Made specifically for an ipad, orginally made by https://github.com/JohnEarnest/Decker

## How to Use

1. git clone repo
2. Open WigglyPaintWebView.xcodeproj in Xcode
3. Connect ipad to build the app directly
4. Test out the by simply opening WigglyPaintWebView/Resources/wigglypaint.html
